# 📌프로젝트명 Enjoy

프로젝트 소개 OTT 플랫폼을 기획, 디자인, 구현한 웹사이트입니다.

## 💡주요 기능

- 검색 기능 추가
- TV, 영화 메뉴로 구성하여 사용자가 콘텐츠 찾기에 용이
- 장르, 평점, 관람이용가 제공

## 📅 제작 기간

- 7/5 ~ 7/20 (약 2주 소요)

## ✅기술요점

- react를 사용하여 react-Hooks를 이용하여 구현
- axios로 API를 받아와서 사이트 구현
- styled-components를 이용하여 스타일 지정
- Nodejs 언어를 사용하여 npm 설치
- 반응형 작업

## 📝사용언어

자바스크립트 / Nodejs
React
